# Sistema de Segunda Ordem

Este projeto é uma página web que realiza cálculos para sistemas de segunda ordem, com base na entrada de dois parâmetros: a frequência natural (Wn) e o fator de amortecimento (Zeta). A página calcula e exibe os seguintes valores:

- Wd (frequência amortecida)
- Sigma (constante de amortecimento)
- Tempo de Pico
- Tempo de Subida
- Tempo de Acomodação (5% e 2%)
- Máximo Sobressinal (%)

Além disso, os polos do sistema são plotados no plano complexo.

## Estrutura do Projeto

- `index.html`: Arquivo HTML principal que define a interface da aplicação.
- `styles.css`: Arquivo CSS responsável pela estilização da página.
- `script.js`: Arquivo JavaScript que realiza os cálculos e manipula a renderização do gráfico de polos.

## Como usar

1. Insira os valores para Wn (frequência natural) e Zeta (fator de amortecimento).
2. Clique no botão **Calcular**.
3. O resultado será exibido abaixo com todos os parâmetros calculados, e os polos serão plotados no plano complexo.

## Requisitos

- Navegador compatível com HTML5, CSS3 e JavaScript.

## Exemplo

Com os valores:
- **Wn**: 5
- **Zeta**: 0.7

O resultado incluirá os valores calculados, e os polos serão exibidos visualmente no gráfico.
